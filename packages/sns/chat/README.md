# Rsbuild Project

## 1 - Setup

Switch node version with nvm:

```bash
nvm use
```

Install the dependencies:

```bash
bun install
```

## 2 - ENV

```bash
cp .env.example .env
```

set PUBLIC_CHAT_API_ENDPOINT to your chat api endpoint

## 3 - Get Started

Start the dev server:

```bash
bun dev
```

Build the app for production:

```bash
bun build
```
